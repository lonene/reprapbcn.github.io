Welcome to the Downloads Page.

Here you'll find all the files to work with your BCN3D+. We decided to make our documentation even more available, that's why we've created this Downloads Page inside a GitHub repository (a.k.a repo). Even more, every file is in a repository so you can keep track of any change and be up to date.

If you don't know yet about this technology, take a look at this. Our project is Open Source so feel free to download anything you need and help us improve along the way.

Be part of the community!

Currently we support the repos listed in the table below. There are a name and two icons per line. The name will show you all the information in this page. The first icon will take you to the repository in GitHub and the second one lets you download all the data directly in a .zip

Print Configurations // For Slic3r and Cura

Based on our experience in 3D Printing, we've made some base configurations in order to make it easier to print good quality objects from day one.

You will find separated configurations for the single extruder and the dual extruder option. The corresponding files are organized as follows: 

		      		
		Configurations
		├── Single Extruder
		│   ├──Slic3r
		│   ├──Cura	
		└── Dual Extruder
		    ├──Slic3r
		    └──Cura

Note the name of the configuration file!
It contains the material, the initials and the version. Configuraciones.txt contains explanation for every config.

Firmwares // Adapted Marlin to our BCN3D+

In order to give our 3D Printer the intelegence needed to make awesome objects, you need to upload the latest firmware. We currently give support to BCN3D+ firmwares. The standard and the dual extruder.

Even thought we don't support older firmwares, you are still able to download the BCN3D version for all those firsts BCN3D out in the wild.

		      		
		Firmwares
		├── BCN3D
		├── BCN3D+
		└── Dual Extruder

Before uploading a new firmware be sure to follow all the steps and indications. Also remember to install the U8glib in you Arduino libraries!

CAD // From the solidworks file to the STL

RepRapBCN is commited to the global Open Source Project REPRAP. It means you can download all the source files, even the designs of the mechanical pieces that compose the printer. The objective is to get faster and better feedback from the community using our printers. If you detect any error or have any improvements, please let us know by creating an Issue in the GitHub Repository.

You will find folders for the BCN3D+, the dual extruder and pretty soon other upgrades. In each folder there is a BOM (bill of materials), a complete Assembly and the printed parts.

Github visualization

- Check this out! - Now directly from your browser, you can preview the STL file in 3D and keep track of all the changes. Github is pretty awesome huh?

Manuals // Get your allen keys ready

We've made some pretty awesome manuals to help you assembling the BCN3D+. We have two versions, the PDF and the 3DVIA. All the manuals are based on the 3D model of the printer, so you'll be able to see 3D assemblies with all the parts and pieces.

Again, the manuals are supposed to guide you in the process of assembling the printer. If you detect any mistake or improvement, please let us know by submitting and Issue in the git repo.

The good stuff comes when you use the 3DVIA manuals. An executable application lets you rotate, zoom and select the parts in a 3D dimensional way. Easy as pie!

SD Workshop // All the SD content in one place

If you are attending one of our Workshops and you prefer to download the latest documents or your SD card is outdated, just download the .zip and you're ready to go. It contains everything to follow the workshop and get your 3D printer up and running
